# Coffee Sales Analytics Dashboard | Power BI

## Problem Statement
Businesses require real-time insights into sales performance to optimize operations and improve decision-making. This project focuses on analyzing coffee shop sales data through an interactive Power BI dashboard to identify revenue trends, customer purchasing behavior, and peak sales periods.

---

## Objectives
- Analyze transaction-level sales data to evaluate overall revenue performance  
- Identify peak sales hours and high-demand periods  
- Compare weekday and weekend sales trends  
- Evaluate product-level demand across different coffee categories  

---

## Dataset
- Coffee Shop Sales Dataset  
- Features include: Date, Time, Product, Sales, Transaction ID  
- Data represents daily transactions across multiple time periods  

---

## Tools and Technologies Used
- Power BI  
- Power Query  
- DAX  
- Microsoft Excel  

---

## Approach
- Performed data cleaning and transformation using Power Query  
- Created calculated measures using DAX (Total Revenue, Orders, Average Sales)  
- Built interactive dashboard with filters and slicers  
- Designed visualizations to analyze time-based and product-level trends  

---

## Key Insights
- Total Revenue: ₹112K  
- Peak Sales Hour: 10 AM, indicating strong morning demand  
- Weekdays generated higher revenue compared to weekends  
- Latte and Cappuccino recorded the highest sales volumes  

---

## Dashboard Features
- KPI cards displaying key metrics such as total revenue, cups sold, and transaction count  
- Time-based visualizations for hourly and daily sales patterns  
- Product category analysis to compare demand across coffee types  
- Interactive filters for dynamic exploration  

---

## Business Impact
- Helps optimize inventory planning based on high-demand products  
- Supports staffing decisions during peak hours  
- Enables better pricing and promotional strategies  
- Improves overall operational efficiency through data-driven insights
